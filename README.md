# Codex Online

## Cíl projektu
Cílem projektu bylo vytvořit webovou implementaci inspirovanou deskovou hrou **Codex Naturalis**, která umožňuje více hráčům hrát přímo v internetovém prohlížeči na jednom zařízení.

## Postup
Projekt byl vytvořen pomocí **HTML, CSS a JavaScriptu** bez použití externích frameworků. Aplikace byla rozdělena do několika modulů pro správu hry, vykreslování herní plochy, pravidla a výsledky. Během vývoje bylo implementováno pokládání karet pomocí Drag & Drop principu, animace rozhraní a systém pro výpočet bodů.

## Popis funkčnosti
Aplikace obsahuje několik částí — menu, nastavení hry, pravidla, herní plochu a výsledky. Hráči mohou zadat svá jména, následně pokládat karty na herní plochu a získávat body podle pravidel hry.  Výsledky her jsou ukládány pomocí **LocalStorage** a navigace mezi jednotlivými částmi aplikace využívá **History API** bez nutnosti reloadu stránky.
